Quotebot - Stock quotes to the IRC
==================================

Quotebot shows stock quotes in IRC chat. Graphs from Yahoo! Finance are
included.

Usage
-----

Install with `npm install -g quotebot` and then run
`quotebot [host] [channels]`.
